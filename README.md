# fitr-server

Express/Mongodb based project.

## DEVELOPMENT
1ยบ - `npm install`

2ยบ - Instalar o [mongo](https://www.mongodb.com/) e:
```
$ mongod & (abre a :27017 sem bloquear o terminal)
$ mongodb (acessa o repl)
$ > use fitr  (cria o db fitr)
```
3ยบ - `npm start`
